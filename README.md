## Task1 安裝SDK(開發工具箱)

[.NET 10.0](https://dotnet.microsoft.com/zh-tw/download)

[自學 .NET 教學課程 - 5 分鐘的 Hello World](https://dotnet.microsoft.com/zh-tw/learn/dotnet/hello-world-tutorial)

[練習 - 撰寫您的第一個程序代碼](https://learn.microsoft.com/zh-tw/training/modules/csharp-write-first/2-exercise-hello-world)

程式ex1.cs
```cs
Console.WriteLine("Hello World!");
```
執行
```cmd
C:\myfiles>dotnet run ex1.cs
Hello World!
```
